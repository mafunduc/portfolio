Sales Stage Framework Implementation

The Problem: Three different opportunity types (Net-New, Expansion, Renewal) all forced through the same generic stages, creating forecasting confusion and process inefficiency.

Solution

Designed differentiated stage frameworks:

  • Net-New: 11-stage process from SAL through closed won with clear exit criteria
  • Expansion: 5-stage streamlined process for existing customers
  • Renewal: 6-stage process with 120-day automated triggers
  • Mapped all stages to forecast categories (Pipeline, Upside, Most Likely, Commit)
  • Built "stale opportunities" monitoring for deals stuck >30 days
Business Impact: Currently implementing - projected to improve forecast accuracy and pipeline velocity by creating clear progression criteria
